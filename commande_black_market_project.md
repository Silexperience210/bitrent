# Projet : Commande Black Market

Ce projet vise à développer une application de commandes pour le marché noir.

## Objectifs principaux :
1. Créer une interface utilisateur pour les commandes
2. Gérer la connexion sécurisée aux serveurs
3. Implémenter un système d'authentification avancée
4. Développer des fonctionnalités de suivi des commandes

## Technologies utilisées :
- Python 3.x
- Flask/Django pour le backend
- React/Vue.js pour l'interface frontend
- PostgreSQL/MySQL pour la base de données
- JWT pour l'authentification sécurisée

## Étapes de développement :
1. Création du modèle de données
2. Développement du backend API
3. Création de l'interface utilisateur
4. Implémentation des fonctionnalités de suivi
5. Tests unitaires et intégrés

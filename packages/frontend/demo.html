🚀 How It Works

1. Connect Wallet

Sign in with your Nostr wallet (Alby, nos2x, or similar)

2. Browse Miners

See available mining rigs with real-time hashrate and pricing

3. Select Duration

Choose rental duration (minutes to hours) and see instant pricing

4. Receive Invoice

Get Lightning invoice for your rental

5. Pay Instantly

Send sats via Lightning Network using NWC

6. Mining Active

Miner starts mining to your address immediately

📚 API Endpoints

Open RESTful API for miners, rentals, and payments:

GET /api/mineurs
List all available miners with status and pricing
POST /api/auth/challenge
Get Nostr auth challenge for signing
POST /api/auth/verify
Verify signed challenge and get JWT token
POST /api/rentals
Create new rental (requires auth token)
POST /api/payments
Create payment invoice for rental
GET /api/health
System health check with component status
